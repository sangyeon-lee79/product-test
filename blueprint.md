# Blueprint: 방울아놀자 (Bang-ul Play)

## 1. 프로젝트 개요

'방울아놀자'는 반려동물 보호자, 서비스 제공자, 그리고 관리자를 위한 종합 SNS 플랫폼입니다. 보호자는 반려동물의 건강, 활동, 이벤트를 타임라인에 기록하고 공유할 수 있습니다. 서비스 제공자는 자신의 서비스를 홍보하고 예약을 관리하며, 보호자와 소통할 수 있습니다. 관리자는 플랫폼의 핵심 데이터를 관리하고 시스템 전반을 운영합니다.

## 2. 현재 구현된 기능 (As-Is)

### V0.2 (Phase A 완료)
- **UI 프레임워크:** 다크/라이트 모드를 지원하는 기본 UI 셸
- **역할 기반 뷰:** 사용자(User), 공급자(Provider), 관리자(Admin) 역할에 따라 뷰를 전환하는 기능
- **다국어 지원:** 한국어, 영어 등 다국어 전환 기능
- **동적 마스터 데이터 관리 (CRUD):**
    - **카테고리 관리:**
        -   신규 카테고리 추가 (`Create`)
        -   카테고리 목록 조회 (`Read`)
        -   카테고리 활성/비활성 상태 변경 (`Update`)
    - **아이템 관리:**
        -   선택된 카테고리에 신규 아이템 추가 (`Create`)
        -   아이템 목록 조회 (`Read`)
        -   아이템 `item_key` 수정 (`Update`)
        -   아이템 활성/비활성 상태 변경 (`Update`)
- **코드 구조:** 가독성과 유지보수성을 위해 CORE, ADMIN, CRUD 로직으로 코드를 구조화하고 리팩토링을 완료했습니다.
