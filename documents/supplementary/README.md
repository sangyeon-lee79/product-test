# 단계별 보충자료 패키지 (s0.1 ~ s12)

요청 기준: `documents/PLAN.md`의 S0-1~S12를 기준으로 단계별 실행에 필요한 보충자료를 정리했습니다.

## 폴더 구성
- `s0.1/` Repo & 폴더 구조
- `s0.2/` 인프라 파이프라인
- `s0.3/` 통신 검증
- `s1/` 언어관리
- `s2/` 마스터 데이터
- `s3/` 질병 연결 매핑
- `s4/` 국가/통화
- `s5/` 인증 + 스토리지
- `s6/` Guardian 프로필 + 펫
- `s7/` 질병 기록 + 타임라인
- `s8/` 오프라인 동기화
- `s9/` Provider 매장 + 서비스
- `s10/` 예약 + 완료사진 + 바이럴
- `s11/` 광고 + 통계
- `s12/` OAuth 전환 + 마무리

각 폴더 `README.md`에는 아래를 포함합니다.
- 단계 목표
- 필요한 보충자료(체크리스트)
- 현재 저장소 기준 참조 파일
- 바로 실행 가능한 준비 항목
