# S11 보충자료 - 광고 + 통계 대시보드

## 1. 광고 슬롯 정의서 (Ad Slots)
- **`feed_list_banner`**: 피드 목록 중간에 삽입되는 배너 광고.
- **`store_detail_banner`**: 매장 상세 페이지 하단에 삽입되는 배너 광고.
- **`profile_banner`**: (선택) 사용자 프로필 하단 배너.

---

## 2. `no_health_zone` 정책
- **정의**: 신뢰도 유지를 위해 건강/질병 관련 화면에서는 광고를 절대 노출하지 않는다.
- **대상 화면**: 질병 기록 입력(`log_type`), 타임라인(`S7`), 펫 질병 관리 화면.
- **구현**: 앱 내 광고 로더가 현재 화면의 `no_health_zone` 속성을 확인하여 광고 요청을 차단한다.

---

## 3. 광고 노출 카운트 집계 규칙
- **집계**: 광고가 실제 화면에 일정 시간(예: 1초 이상) 노출되었을 때 `POST /api/v1/ads/impression` 호출.
- **중복 방지**: 동일 사용자/동일 세션 내 과도한 호출은 서버 측에서 필터링하거나 비율로 집계한다.

---

## 4. 대시보드 KPI 정의 (통계)
관리자 대시보드에서 실시간/배치로 집계할 주요 지표:
- **사용자**: 신규 가입자 수(Guardian/Provider), 활성 사용자 수(DAU).
- **콘텐츠**: 생성된 피드 수, 좋아요/댓글 총합.
- **비즈니스**: 예약 요청 및 완료 건수, 평균 평점(선택).
- **수익화**: 광고 노출 수(Impression) 및 슬롯별 클릭률(CTR).
